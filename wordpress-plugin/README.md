# Oppijan Korp WordPress Plugin

This directory contains the WordPress plugin version of Oppijan Korp.

## Structure

```
wordpress-plugin/
├── oppijan-korp.php    # WordPress plugin file
└── dist/               # Built React app (generated by npm run build:wordpress)
    ├── assets/         # JS, CSS, and image assets
    └── .vite/          # Vite manifest for asset loading
```

## Installation

### Manual Installation
1. Copy this entire `wordpress-plugin` directory to your WordPress plugins folder:
   ```bash
   cp -r wordpress-plugin /path/to/wordpress/wp-content/plugins/oppijan-korp
   ```

2. Activate the plugin in WordPress admin

3. Use the shortcode `[oppijan-korp]` in any page or post

### Build Process

The `dist/` folder is automatically generated when you run:
```bash
npm run build:wordpress
```

This builds the React app with WordPress-specific configuration:
- Mount element ID: `oppijan-korp-app`
- Base path: `/wp-content/plugins/oppijan-korp/dist/`
- Output directory: `wordpress-plugin/dist/`

## Environment Configuration

WordPress-specific settings are defined in `.env.wordpress`:
- `VITE_MOUNT_ELEMENT_ID=oppijan-korp-app` - Must match the div ID in oppijan-korp.php
- `VITE_IS_WORDPRESS=true` - Flag for WordPress-specific code
- Add custom WordPress settings as needed

## Usage

After activating the plugin, add the shortcode to any WordPress page:

```
[oppijan-korp]
```

The React app will be loaded only on pages that use this shortcode.
